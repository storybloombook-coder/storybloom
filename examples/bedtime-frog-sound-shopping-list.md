# Sound Shopping List — "Pip and Posy: The Bedtime Frog"

Every sound THIS book needs, so it can run fully on a bundled library with NO
AI-generated audio. Gather these from free, commercial-OK sources before/at the
session. Sources: Pixabay (pixabay.com/sound-effects), Mixkit
(mixkit.co/free-sound-effects), Freesound (freesound.org — prefer CC0).

Keep ambient beds 10-30s and loopable; effects 1-3s; voices short stings.
File names below are SUGGESTED — match them to the ids so the app finds them.
